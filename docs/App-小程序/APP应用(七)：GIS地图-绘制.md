# GIS地图-地图绘制

### 地图绘制

- 绘制功能可以采用mapbox框架的mapbox-gl-draw绘制功能（打包版可以从网上找到），将绘制功能打包至本地动态引用。也可以自己写一套绘制功能

- mapbox-gl-draw如果需要自定义UI是很麻烦的，但是扩展方法可以从mapbox的github仓库中查找。

- 可以在打包一个本地版的 [turf.js](hhttps://turfjs.org/)，来搭配点线面的计算
